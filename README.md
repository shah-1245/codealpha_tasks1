# Calculator

## Project Description
This is a simple Calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides a clean, responsive interface with keyboard support.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear screen
- Delete last character
- Keyboard support
- Responsive design
- Button hover effects

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

Calculator/
│── index.html
│── style.css
│── script.js

## How to Run
1. Download or clone the project.
2. Open `index.html` in any web browser.
3. Or run using VS Code with the Live Server extension.

## Keyboard Shortcuts
- Numbers (0–9)
- +, -, *, /
- Enter = Calculate
- Backspace = Delete last character
- Escape = Clear screen
